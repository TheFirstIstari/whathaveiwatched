# Deploy Steps (reference)

## SpacetimeDB Module

**Status:** ✅ Published (commit `3d6fcc4`, 2026-06-13)

### To update the module after local changes:

```bash
# 1. Rsync source to server
rsync -avz --delete ~/Documents/ObsidianVault/Programming/SideProjects/IHaveWatched/module/src/ root@100.82.166.71:/root/IHaveWatched-module/src/

# 2. SSH and publish
ssh root@100.82.166.71
cd IHaveWatched-module
spacetime build
spacetime publish whathaveiwatched --server http://127.0.0.1:3000 --yes --bin-path target/wasm32-unknown-unknown/release/ihavewatched_module.wasm
```

## Frontend

**Status:** ✅ Auto-deployed via GitHub Actions (push to main triggers CI)

The CI workflow:
1. SSHes to the server via Tailscale
2. Rsyncs the frontend code to `/opt/whathaveiwatched/`
3. Runs `npm ci && npm run build && pm2 restart whathaveiwatched`

### Manual deploy (if CI is down):

```bash
rsync -avz --delete --exclude node_modules --exclude .next ~/Documents/ObsidianVault/Programming/SideProjects/IHaveWatched/frontend/ root@100.82.166.71:/opt/whathaveiwatched/
ssh root@100.82.166.71 "cd /opt/whathaveiwatched && npm ci && npm run build && pm2 restart whathaveiwatched"
```
